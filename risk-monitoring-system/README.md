# AI风险监控系统 (C2C)

基于Cursor AI生成的一个完整的C2C交易平台风险监控仪表板前端实现。

## 🎯 项目概述

这是一个现代化的AI风险监控系统，专为C2C交易平台设计，用于实时监控用户风险行为、分析交易模式并提供智能风险评估。

## 📦 版本说明

本项目包含两个版本：

### 🔹 V1 版本 (index.html)
**定位**: 基础风险监控系统
- 适合小型团队和基础风险管理
- 功能完整，界面简洁
- 快速上手，学习成本低

### ⭐ V2 版本 (index-v2.html) - **推荐使用**
**定位**: 高可用AI风险监控系统 - 运营优化版
- 专为专业运营团队打造
- 5大专业视图切换（概览/用户/事件/关系图/操作台）
- 实时告警中心 + Top风险排行
- 可视化数据分析（ECharts图表）
- 批量操作功能（提升效率10倍）
- 团伙网络监控
- 任务管理系统
- 专业运营指南

## ✨ 核心功能

### 📊 风险监控仪表板
- **实时风险用户列表** - 按风险评分排序显示高风险用户
- **多维度过滤器** - 支持风险等级、类型、日期范围、用户ID等多种筛选条件
- **分页显示** - 高效处理大量数据，支持分页浏览

### 🔍 详细风险分析
- **用户风险详情** - 完整的用户信息和风险事件分析
- **设备关联分析** - 多设备使用行为追踪
- **IP地址监控** - IP位置和变更频率分析
- **支付方式追踪** - 支付QR码和账户共享检测
- **交易行为分析** - 交易频率、取消率、金额统计

### 🤖 AI智能评分
- **风险评分算法** - 基于多因子AI分析的综合评分
- **风险等级分类** - 高/中/低风险自动分类
- **关键风险因素** - 突出显示主要风险触发点

### 📈 关系图分析
- **设备关系图** - 可视化多设备之间的关联
- **IP关系图** - 显示IP地址共享情况
- **支付关系图** - 追踪支付方式的关联账户

### 📋 用户资料面板
- **账户信息** - 注册时间、交易统计、活跃度
- **行为标签** - 用户行为模式分析
- **安全标签** - 安全风险标识

### 📝 审计日志
- **操作记录** - 所有风险监控操作的详细日志
- **操作员追踪** - 记录操作人员和时间
- **日志管理** - 支持日志清理和导出

## 🛠️ 技术栈

- **HTML5** - 语义化结构和现代Web标准
- **CSS3** - 现代化样式，响应式设计，支持暗色主题
- **Vanilla JavaScript** - 无框架依赖，纯原生实现
- **Font Awesome** - 专业图标库
- **Chart.js** - 图表可视化 (预留接口)

## 🚀 快速开始

### 本地运行

1. **克隆项目**
   ```bash
   cd /path/to/risk-monitoring-system
   ```

2. **启动服务器**
   ```bash
   python3 -m http.server 8000
   # 或使用其他HTTP服务器
   ```

3. **访问应用**
   
   **V1 基础版**:
   ```
   http://localhost:8000/index.html
   ```
   
   **V2 运营优化版** (推荐):
   ```
   http://localhost:8000/index-v2.html
   ```

### 功能演示

1. **浏览风险用户列表**
   - 查看按风险评分排序的用户列表
   - 观察风险等级颜色标识 (红/橙/绿)

2. **使用过滤器**
   - 选择风险等级 (高/中/低风险)
   - 选择风险类型 (设备/IP/交易/支付/团伙)
   - 设置日期范围
   - 搜索特定用户ID

3. **查看用户详情**
   - 点击表格行或"查看"按钮
   - 右侧边栏显示完整风险分析
   - 查看用户资料、AI评分、关系图

4. **探索关系图**
   - 选择不同的图表类型 (设备/IP/支付)
   - 观察账户间的关联关系

5. **审计日志**
   - 查看所有风险监控操作记录
   - 支持日志导出功能

### API接口设计
- **GET /api/risk-users** - 获取风险用户列表
- **GET /api/risk-detail/{userId}** - 获取用户风险详情
- **GET /api/relationships/{type}/{userId}** - 获取关系图数据
- **GET /api/audit-logs** - 获取审计日志
