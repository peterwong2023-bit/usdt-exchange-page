### 7. 交互功能总结

**跳转联动**：
- 点击Risk Summary的Top3卡片 → 滚动到规则 + 高亮3秒 + 自动展开
- 点击时间轴事件 → 滚动到规则 + 高亮3秒 + 自动展开
- 点击层级Top贡献标签 → 滚动到规则 + 高亮
- 点击安全标签 → 跳转到对应规则

**展开/收起**：
- 用户画像：展开全部/收起
- 规则卡片：点击展开/收起
- Tab切换：证据/计算/建议
