## 业务管理 — 模块总览

```
VVOSS Admin
│
├── 业务管理（01-business-management.md）
│   ├── 1. 平台管理 — 最顶层租户，管理多个商户
│   ├── 2. 商户管理 — 业务实体，配置风控/归集/出款
│   │   └── 客户管理 — 商户的终端用户（同页面 Tab）
│   ├── 3. 激活平台 — TRON 钱包激活树状结构管理
│   └── 4. 能量平台 — TRON 能量/带宽供应商管理
│
├── 资金&交易（02-transaction-funds.md）
│   ├── 1. 钱包管理 — 多链多币种钱包，含流水查询
│   ├── 2. 链上交易 — 链上交易记录查询
│   ├── 3. 出款记录 — 出款详情 + 任务链路依赖
│   ├── 4. TRON委托管理 — 能量/带宽委托
│   ├── 5. TRON质押管理 — TRX 质押生命周期
│   ├── 6. TRON投票管理 — 超级代表投票 + 奖励领取
│   ├── 7. 转账费用日志 — Gas 消耗明细 + 汇总统计
│   ├── 8. 统计报表 — 商户维度资金汇总
│   └── 9. 链上余额查询 — 多链余额同步工具
│
├── 风控&合规（待整理）
│   ├── 地址标签
│   ├── 风控记录
│   └── ...
│
└── 后台&管理（待整理）
    ├── 管理员
    ├── 操作日志
    ├── 系统配置
    └── ...
```
