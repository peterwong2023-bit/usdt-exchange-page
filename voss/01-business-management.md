# VVOSS Admin — 业务管理模块脑图

> 最后更新: 2026-03-17 | 审查范围: 后台管理系统
>
> 系统层级: 平台 → 商户 → 客户 → 钱包
>
> 币种标识: KHR / KHRX / KHRD / KHRDOLD / KHRCOLD / KHRE / KHRC

---

## 1. 平台管理

```
平台管理
├── 列表页
│   ├── 筛选：平台名称 / 平台代码 / 搜索+重置
│   ├── 表格列
│   │   ├── 平台ID（自增）
│   │   ├── 平台名称
│   │   ├── 平台代码（唯一标识，如 ssg、liveplatform）
│   │   ├── 状态（启用/禁用）
│   │   ├── 创建时间
│   │   └── 操作：查看 / 编辑
│   ├── 右上角「创建平台」
│   └── 分页（共 10 条）
│
├── 创建平台弹窗
│   ├── * 平台名称
│   └── * 平台代码（如 btcplatform）
│
├── 平台详情弹窗
│   ├── 基本信息
│   │   ├── 平台ID / 平台代码 / 平台名称
│   │   ├── 状态（启用/禁用）
│   │   ├── 创建时间 / 更新时间
│   │   └── 编辑按钮
│   └── 平台统计
│       ├── 商户数量
│       ├── 钱包数量
│       └── 活跃商户
│
└── 概念说明
    └── 平台是 VOSS 最顶层租户，一个平台下管理多个商户
```

## 2. 商户管理

```
商户管理

│   │   ├── 轮换策略（如 手动更新）
│   │   ├── 归集阀值（如 5U、10U，未设置显示 --）
│   │   ├── 自动归集（启用/内部/--）
│   │   ├── 余额(多链)（KHR/KHRX/KHRD 等，+N 展开更多）
│   │   ├── 异常入款（外部资金直接打到归集地址的笔数）
│   │   ├── 状态（启用/停用）
│   │   ├── 创建时间
│   │   └── 操作：查看 / 归集
│   ├── 右上角「+ 添加商户」
│   └── 分页（共 267 条）
│
├── 添加商户弹窗
│   ├── * 商户名称
│   ├── * 所属平台（下拉）
│   ├── 钱包轮换策略
│   │   └── * 策略类型（下拉，如 手动更新）
│   ├── 归集策略
│   │   ├── 自动归集（开关，默认 OFF）
│   │   ├── KHR归集阀值（必须 > 0）
│   │   └── 闲置归集（开关，默认 OFF）
│   ├── 回调URL（可选，接收钱包事件回调通知）
│   └── ☑ 同步创建并关联归集商户（默认勾选）
│
├── 编辑商户弹窗
│   ├── * 商户名称
│   ├── * 平台（只读）
│   ├── 回调URL
│   ├── * 归集商户（下拉关联）
│   ├── 激活商户（下拉）
│   ├── 能量商户（下拉）
│   ├── 风险归集商户（下拉）
│   ├── 能量平台（下拉）
│   ├── * 状态（启用/禁用）
│   ├── 启用出款API（开关）
│   ├── 启用出款回调（开关）
│   └── 最大API出款金额 (KHR)
│
├── 商户详情弹窗（5 个 Tab）
│   │
│   ├── Tab 1: 基本信息
│   │   ├── 商户ID / 商户名称
│   │   ├── 平台 / 状态（启用 + 助记词已初始化）
│   │   ├── API密钥（可复制）
│   │   ├── 归集商户（关联 ID）
│   │   ├── 风险归集（关联 ID）
│   │   ├── 激活商户（关联 ID）
│   │   ├── 能量商户 / 能量平台
│   │   ├── 轮换策略（可编辑）
│   │   ├── 归集策略（如 5U / 启用，可编辑）
│   │   ├── 推送配置（可编辑）
│   │   ├── 回调URL
│   │   ├── 出款API / 出款回调（启用/禁用）
│   │   ├── API出款限额（如 200 KHR）
│   │   ├── 风控配置摘要（评分阈值 / 隔离入款 / 拦截出款）
│   │   └── 创建时间 / 更新时间
│   │
│   ├── Tab 2: 钱包策略
│   │   ├── 轮换策略
│   │   │   └── 策略类型（下拉，如 手动更新）
│   │   └── 归集策略
│   │       ├── 自动归集（开关）
│   │       ├── KHR归集阀值
│   │       ├── 闲置归集（开关）
│   │       ├── 闲置归集最小阀值
│   │       ├── 可用能量最小笔数
│   │       └── 闲置归集冷却时间
│   │
│   ├── Tab 3: 访问凭证
│   │   ├── 「+ 新增密钥」
│   │   └── 密钥列表
│   │       ├── Access ID
│   │       ├── 名称
│   │       ├── 状态（激活）
│   │       ├── 创建时间 / 最后使用时间
│   │       └── 操作：获取密钥 / 删除
│   │
│   ├── Tab 4: 风控配置
│   │   ├── 总开关（开启后对所有交易进行风险检测）
│   │   ├── 风险评分阈值 (0-100)
│   │   │   └── ≥ 阈值 → 触发风险管理及拦截
│   │   ├── 入款风控配置 [隔离入款]
│   │   │   ├── 风控提供商（金额范围可重叠，多重检测）
│   │   │   │   ├── Bitrace — 开关 + 检测金额范围 (KHR)
│   │   │   │   └── Chainalysis — 开关 + 检测金额范围 (KHR)
│   │   │   ├── 入款检测范围预览（可视化条形图）
│   │   │   └── 隔离入款开关（高风险入款隔离，不归集，可退回）
│   │   └── 出款风控配置 [拦截出款]
│   │       ├── 风控提供商（同入款结构）
│   │       │   ├── Bitrace — 开关 + 检测金额范围
│   │       │   └── Chainalysis — 开关 + 检测金额范围
│   │       ├── 出款检测范围预览
│   │       └── 拦截出款开关（高风险出款自动拒绝）
│   │
│   └── Tab 5: 更多操作
│       ├── 数据查询
│       │   ├── 入款明细
│       │   └── 待退回记录
│       └── 批量操作
│           └── 批量更换钱包
│
└── 手动归集弹窗（从列表页「归集」按钮触发）
    ├── 筛选：链（波场）/ 资产类型（KHR）/ 余额大于 / 有风险
    ├── 汇总信息
    │   ├── 商户 / 链+代币
    │   ├── 归集钱包数 / 归集总金额
    │   ├── 当前平台能量（可支持 N 笔交易）
    │   ├── 当前平台带宽（可支持 N 笔交易）
    │   └── 手动归集配置
    ├── 钱包列表：钱包ID / 钱包地址 / 可用余额 / 冻结余额 / 状态
    └── 操作：取消 / 确认归集
```

## 3. 客户管理

```
客户管理（与商户管理同页面，顶部 Tab 切换）
├── 列表页
│   ├── 筛选：选择平台 / 选择商户 / 外部客户ID / 查询+重置
│   ├── 表格列
│   │   ├── ID（系统内部客户 ID）

│   │   ├── 商户

│   │   └── 操作：查看
│   └── 分页（共 3223 条）
│
├── 客户详情弹窗
```

## 4. 激活平台

```
激活平台（管理 TRON 链上钱包激活的树状结构）
├── Tab 1: 商户管理
│   ├── 列表页（激活商户管理）
│   │   ├── 表格列
│   │   │   ├── 商户ID / 商户名称
│   │   │   ├── 子节点数
│   │   │   ├── 深度
│   │   │   ├── 最大钱包数量
│   │   │   ├── 创建时间
│   │   │   └── 操作：查看详情
│   │   └── 刷新按钮
│   │
│   └── 激活商户详情页（全屏页面）
│       ├── 配置信息
│       │   ├── 商户ID / 商户名称 / 平台
│       │   ├── 深度 / 每个节点子节点数 / 钱包数量
│       │   ├── 创建时间 / 更新时间
│       │   └── 回收地址 / 回收时间
│       │
│       ├── 激活状态（6 步流水线）
│       │   ├── 生成中 → 生成助记词和创建所有节点钱包
│       │   ├── 待激活 → 节点可接收 KHRX，等待激活
│       │   ├── 分配中 → 分流节点向下分配资金
│       │   ├── 可使用 → 有激活节点接收 KHRX，可配置激活服务
│       │   ├── 回收中 → 正在回收所有节点资金
│       │   └── 已回收 → 所有节点资金已回收完成
│       │
│       ├── 统计信息
│       │   ├── KHRX 总余额 / 根节点地址
│       │   ├── 分流节点总数 / 有KHRX分流节点数
│       │   ├── 分流节点KHRX总数 / 可分配的KHRX总数
│       │   ├── 激活节点总数 / 有KHRX激活节点数
│       │   ├── 激活节点KHRX总数
│       │   └── 总使用激活次数
│       │
│       └── 钱包列表
│           ├── 操作（展开）/ 钱包ID / 编号
│           ├── 地址（可复制+链接）
│           ├── 状态（默认）/ 类型（分流）
│           ├── KHRX余额
│           └── 转入笔数 / 转出笔数
│
└── Tab 2: 生成商户
    ├── 参数配置
    │   ├── * 激活商户（下拉）
    │   ├── * 每人子节点数
    │   └── * 最大深度
    └── 按钮：「生成计划」
```
