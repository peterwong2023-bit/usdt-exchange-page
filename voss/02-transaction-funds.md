## 1. 钱包管理

```
钱包管理

│   ├── 筛选：选择平台 / 选择商户 / 类型 / 钱包地址 / 选择链

│   │   ├── 序号（地址序号）
│   │   ├── 余额/来源（多 Token：KHRX 绿 / KHR 蓝 等）
│   │   ├── 冻结金额/可用金额

│   │   └── 操作：查看 / 更换

├── 添加钱包弹窗
│   ├── * 平台（下拉）
│   ├── * 商户（下拉）
│   └── * 链（下拉）
│
├── 钱包详情弹窗
│   ├── 基本信息
│   │   ├── 钱包ID / 钱包地址（复制 + QR + 链接）
│   │   ├── 钱包类型（客户/商户）/ 链（TRON / ETHEREUM）
│   │   ├── 商户（含 ID）/ 平台
│   │   ├── 状态（已派生）/ 地址序号
│   │   ├── 客户外部ID / 激活状态（待激活）
│   │   ├── 总转入金额 / 总转入笔数
│   │   ├── 总转出金额 / 总转出笔数
│   │   ├── 风险状态（⚠ 有风险）
│   │   ├── 创建时间 / 更新时间
│   │   ├── 最后使用 / 最后风险检查

        ├── ID（流水 ID）
        ├── 操作类型（入款 + 风险代收 等，多标签）
        ├── 货币类型（KHR）
        ├── 操作金额（如 +3、+2.22）
        ├── 操作前余额 / 操作后余额
        ├── 操作冻结金额 / 操作前冻结金额 / 操作后冻结金额
        ├── 备注（交易哈希，可复制）
        └── 创建时间
```
