# IP 白名单测试用例 v1.0

> **关联设计文档**:`IP白名单设计规则_v1.0.md`
> **覆盖范围**:VOSS 端 + 商户端 + 登录校验 + 操作日志
> **版本**:MVP v1.0
> **日期**:2026-04-28

## 用例编号规则

| 前缀 | 模块 |
|---|---|
| `G-TC-xxx` | 通用规则(General) |
| `V-TC-xxx` | VOSS 端(管商户超管) |
| `M-TC-xxx` | 商户端(管子账号) |
| `C-TC-xxx` | 级联规则(Cascade) |
| `L-TC-xxx` | 操作日志(Log) |
| `S-TC-xxx` | 安全 / 异常(Security) |

## 优先级说明

| 标识 | 含义 |
|---|---|
| 🔴 P0 | 阻塞级,主流程必过 |
| 🟡 P1 | 高优,常见路径 |
| 🟢 P2 | 中优,边界 / 异常 |
| ⚪ P3 | 低优,可选增强 |
