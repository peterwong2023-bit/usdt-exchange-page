# 风控配置测试用例

## 配置需求

### 入款检测规则
- **< 100 USDT**：不进行检测
- **100 – 1000 USDT**：使用香港通道（Bitrace）检测
- **> 1000 USDT**：同时使用香港通道（Bitrace）与美国通道（Chainalysis）双重检测

### 出款检测规则
- **1 USDT 起（无上限）**：使用香港通道（Bitrace）检测
- **> 1000 USDT**：同时使用香港通道（Bitrace）与美国通道（Chainalysis）双重检测

## 具体配置步骤（方案1 - Tab切换式）

### 入款配置步骤

1. **启用总开关**
   - 点击页面顶部"风控配置"总开关

2. **切换到入款检测Tab**
   - 点击"入款检测"标签页

3. **配置Bitrace（100-1000 USDT）**
   - 启用Bitrace开关
   - 最小金额：`100`
   - 最大金额：`1000`

4. **配置Chainalysis（> 1000 USDT）**
   - 启用Chainalysis开关
   - 最小金额：`1000`
   - 最大金额：留空（无上限）

5. **验证预览**
   - 查看"入款检测范围预览"
   - 应显示：
     - ⚠️ 0-100: 未检测区域（灰色）
     - 🇭🇰 100-1000: Bitrace独立检测（绿色）
     - 💜 1000+: 双重检测（紫色）

### 出款配置步骤

1. **切换到出款检测Tab**
   - 点击"出款检测"标签页

2. **配置Bitrace（1 USDT起-无上限）**
   - 启用Bitrace开关
   - 最小金额：`1`
   - 最大金额：留空（无上限）

3. **配置Chainalysis（> 1000 USDT）**
   - 启用Chainalysis开关
   - 最小金额：`1000`
   - 最大金额：留空（无上限）

4. **验证预览**
   - 查看"出款检测范围预览"
   - 应显示：
     - ⚠️ 0-1: 未检测区域（灰色）
     - 🇭🇰 1-1000: Bitrace独立检测（绿色）
     - 💜 1000+: 双重检测（紫色）

5. **保存配置**
   - 点击页面底部"保存配置"按钮
   - 等待成功提示

## 具体配置步骤（方案3 - 统一开关）

### 使用分离配置模式

1. **启用总开关**
   - 点击页面顶部"风控配置"总开关

2. **启用"区分入款/出款配置"开关**
   - 点击"区分入款/出款配置"开关
   - 界面将展开显示入款和出款两个独立配置区域

3. **配置入款检测**（步骤同方案1）

4. **配置出款检测**（步骤同方案1）

5. **保存配置**

## 注意事项

1. **金额精度**：测试时注意小数点精度，建议支持2位小数
2. **边界条件**：重点测试以下关键边界值
   - 入款：100 USDT 和 1000 USDT
   - 出款：1 USDT 和 1000 USDT
3. **未检测区域**：
   - 入款：< 100 USDT 不检测
   - 出款：< 1 USDT 不检测
4. **香港通道覆盖范围**：
   - 入款：100-1000 USDT（独立）+ 1000+ USDT（双重）
   - 出款：1 USDT 起全范围覆盖（1-1000独立 + 1000+双重）
5. **双重检测逻辑**：> 1000 USDT 时必须同时调用两个检测通道
6. **配置独立性**：入款和出款配置互不影响，可以有不同的规则
